La mediciones empezaron en diciembre del 2012 y terminaron en marzo del 2015. De la gráfica podemos identificar en la mitad del año, entre los meses de junio y agosto la temperatura del oceano baja. Para observar de manera más clara esta tendencia haremos una grafica para cada una de las islas

Gráficas por Isla#

data['Date'] = pd.to_datetime(data['Date'], format='%Y/%m/%d')
# Agrupar por 'Site' y 'Date' para calcular el promedio de temperatura diario
daily_avg_temp = data.groupby(['Site', 'Date'])['Temp (°C)'].mean().reset_index()

# Obtener una lista de sitios únicos
sites = daily_avg_temp['Site'].unique()

# Generar un gráfico para cada sitio
for site in sites:
    site_data = daily_avg_temp[daily_avg_temp['Site'] == site]
    plt.figure(figsize=(10, 3))
    
    # Graficar los datos
    plt.plot(site_data['Date'], site_data['Temp (°C)'], marker='o',markersize=2)
    plt.title(f'Promedio diario de temperatura - {site}')
    plt.xlabel('Mes y Año')
    plt.ylabel('Temperatura (°C)')
    
    # Configurar el formato del eje x para mostrar meses y años
    plt.gca().xaxis.set_major_formatter(mdates.DateFormatter('%b %Y'))
    
    # Ajustar el intervalo de los ticks para que muestren los meses
    plt.gca().xaxis.set_major_locator(mdates.MonthLocator())
    
    # Rotar las etiquetas del eje x para mejor legibilidad
    plt.xticks(rotation=45)
    
    plt.grid(True)
    plt.tight_layout()
    plt.show()

Después de ver los datos de cada una de las islas podemos identificar que se cumple la tendencia de que en los meses de la mitad del año la temperatura baja y en los extremos aumenta. Tambien es posible evaluar si existen tendencias o cilos relacionados con la hora y la profudidad a la que se toma la medida
